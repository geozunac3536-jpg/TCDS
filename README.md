# Teoría Cromodinámica Sincrónica (TCDS)

Bienvenido al repositorio oficial de la Teoría Cromodinámica Sincrónica (TCDS).

Este proyecto documenta y desarrolla la TCDS, una nueva perspectiva sobre la dinámica de sistemas complejos, con especial foco en aplicaciones científicas y tecnológicas.

## Documentación

La documentación completa está disponible en la carpeta `docs/` y se publica automáticamente con MkDocs y GitHub Pages.

## Estructura del repositorio

- `docs/`: Documentación principal en formato Markdown.
- `mkdocs.yml`: Configuración de MkDocs.
- `.github/workflows/gh-pages.yml`: Automatización para publicar la documentación.
